一个基于 Python/PySide6 的桌面番茄钟挂件，拥有可爱的角色动画和高度可自定义的交互体验。

## ✨ 特性 (Features)

*   **独立的计时器窗口**: 计时器与角色窗口分离，可自由拖拽到屏幕任意位置，互不遮挡。
*   **自适应布局**: 计时器背景随字体大小自动缩放，角色窗口大小可独立调节。
*   **沉浸式动画**: 包含待机、工作、休息、暂停等多种状态的动画反馈。
*   **丰富的语音反馈**: 支持开始、结束、休息、继续、退出等多种场景的随机语音/音效。
*   **便捷的交互**: 全局快捷键、鼠标手势支持，无需打开复杂菜单即可完成常用操作。

## 📖 使用方法 (How to Use)

### 1. 窗口移动与缩放
*   **移动角色**: 在人物窗口任意位置（非按钮区）按住 **左键拖动**。
*   **移动计时器**: 鼠标指向计时器，按住 `Ctrl` + **左键拖动**（会自动跟随人物窗口移动，保持相对位置）。
*   **缩放角色**: 鼠标指向人物窗口，按住 `Shift` + **滚动滚轮**。
*   **缩放计时器**: 鼠标指向计时器，按住 `Ctrl` + **滚动滚轮**（字体和背景会同步缩放）。

### 2. 计时控制
*   **开始/暂停**: 
    *   点击右上角的 **开始按钮** (Start)。
    *   暂停状态下，点击屏幕中间的 **继续按钮** (Resume)。
    *   或者使用右键菜单中的选项。
*   **调整时间 (核心功能)**:
    *   **操作**: 鼠标指向计时器，**直接按住左键拖动**。
        *   **水平拖动**: 精细调整（秒/分）。
        *   **垂直拖动**: 快速调整（分）。
    *   **切换模式 (工作/休息)**:
        *   在待机(Idle)或暂停(Paused)状态下，按 `Alt` 键可切换 **工作时长调节** 和 **休息时长调节**。
        *   *工作模式*: 显示正常待机画面。
        *   *休息模式*: 显示休息画面（通常是睡觉或暂停图标），此时调节的是休息时间。
    *   **双击预设**:
        *   双击计时器数字的不同高度区域，可快速应用预设时间（如顶部15分，底部60分等）。

### 3. 右键菜单
*   在人物窗口或计时器上点击 **右键**，可打开功能菜单：
    *   **暂停/继续**: 控制计时。
    *   **置顶**: 切换窗口是否总在最前。
    *   **语音间隔**: 点击切换语音提醒频率（关/10分/15分/30分）。
    *   **退出**: 关闭程序。
    *   **检查更新**: (如果有配置) 检查资源更新。注意：此功能需要系统中已安装 `git`。

## 🛠️ 安装与运行 (Installation)

### 依赖
*   Python 3.8+
*   PySide6

```bash
pip install PySide6
```

### 运行
```bash
python main.py
```
