# Pomodoro Widget

一个基于 Python/PySide6 的桌面番茄钟挂件。

## 📖 使用方法 (How to Use)

### 基本操作
*   **拖动窗口**: 在人物或非按钮区域按住 **左键拖动** 即可移动位置。
*   **开始/暂停**: 左键点击右上角的 **开始按钮**，或在暂停状态下点击中间的 **继续按钮**。
*   **右键菜单**: 在人物身上点击 **右键**，可唤出功能菜单（暂停、退出、设置等）。
    *   *注意：如果找不到菜单，请尝试点击人物的空白区域。*

### 时间调整
*   **调整工作时间**:
    *   按住鼠标左键拖动 **数字时间**。
    *   **水平拖动**: 调整分钟/秒数。
    *   **垂直拖动**: 调整分钟（快速）。
*   **调整休息时间**:
    *   按住键盘 `Alt` 键 + 鼠标左键拖动 **数字时间**。
    *   *此时会预览休息状态的画面。*

### 快捷预设
*   **工作预设**: 双击数字时间的不同高度位置：
    *   **顶部**: 15 分钟
    *   **中上**: 30 分钟
    *   **中下**: 40 分钟
    *   **底部**: 60 分钟
*   **休息预设**: 按住 `Alt` + 双击数字时间：
    *   **顶部**: 5 分钟
    *   **中上**: 10 分钟
    *   **中下**: 15 分钟
    *   **底部**: 启用/禁用 "工作结束自动退出"

### 自定义布局 (Layout Edit)
*   **调整时间位置**: 按住 `Ctrl` + 拖动时间数字。
*   **调整字体大小**: 按住 `Ctrl` + 滚动鼠标滚轮。
