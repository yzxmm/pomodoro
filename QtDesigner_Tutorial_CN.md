# Qt Designer 简明中文教程 (针对番茄钟项目)

这份文档专门为您准备，帮助您快速上手 Qt Designer 并微调番茄钟的右键菜单。

## 1. 界面快速入门 (哪里是哪里？)

启动 Designer 后，界面主要分为三个区域：

*   **左侧：组件箱 (Widget Box)**
    *   这里是仓库，所有的按钮、标签都在这。
    *   *操作*：按住 `Push Button` 拖到中间的画布上，就能加新按钮。

*   **中间：画布 (Form)**
    *   这里是您的设计台，显示当前的 `menu.ui`。
    *   *操作*：点击按钮可以选中；点击按钮之间的空白处可以选择“布局管理器”（用来调间距）。

*   **右侧：属性编辑器 (Property Editor)**
    *   **最重要的地方！** 所有的微调都在这里。
    *   如果您没看到这个窗口，去菜单栏 `View -> Property Editor` 打开。

## 2. 核心操作：如何微调菜单？

### A. 调整按钮的大小和图标
1.  在中间画布上，**点击选中** 您想调整的按钮（例如暂停按钮）。
2.  看右侧的 **属性编辑器**，向下滚动找到以下属性：
    *   **iconSize** (图标大小): 默认是 `24 x 24`。
        *   *怎么改*：点击 `Width` 旁边的数字，输入 `32`，回车。您会发现图标变大了。
    *   **minimumSize** (最小尺寸): 如果觉得按钮太挤，可以设置一个最小宽高，比如 `40 x 40`。

### B. 调整按钮之间的间距 (最常用)
按钮之间挨得太近或太远？
1.  **不要选中按钮**，而是点击按钮之间的**空白区域**（或者点击右侧对象树里的 `ContextMenu`）。
2.  看右侧 **属性编辑器**，滚到最下面，找到绿色背景的 **Layout** 部分：
    *   **layoutSpacing** (内部间距): 按钮之间的距离。默认可能是 6 或 10，改小一点按钮就会凑得更近。
    *   **layoutLeftMargin** / **layoutRightMargin** (边缘留白): 按钮距离窗口边缘的距离。

### C. 调整按钮位置
1.  鼠标按住按钮，直接**拖拽**。
2.  您会看到红色的指示线，松手后按钮就会自动吸附到网格的新位置里。
3.  **跨列显示**（像“检查更新”那样占满一行）：
    *   右键点击按钮 -> `Layout Alignment` (布局对齐) -> 这里的选项一般不用动。
    *   *更简单的方法*：拖拽按钮的边缘（如果不让拖，可能是被布局限制了，通常直接把按钮拖到新的一行即可，它会自动占满）。

## 3. 常见问题与避坑

### Q: 为什么我看不到文字，只有图标？
**A**: 为了所见即所得。
因为我们的代码逻辑是：**“只显示图标，不显示文字”**（除了菜单项）。
我们在 Designer 里把文字清空了，并且去掉了边框，这样您看到的预览图就和实机运行效果一模一样了。

### Q: 我想换个图标怎么办？
1.  选中按钮。
2.  在右侧属性里找 **icon**。
3.  点击右边的小箭头 -> `Choose Resource...`。
4.  在弹出的窗口里，展开左边的 `menu.qrc`，就能看到所有的手绘图了。选中一个点击 OK。

### ⚠️ 高能预警：千万别改这个！
在右侧属性里，有一个叫 **objectName** 的属性（通常在第一行）。
*   `btn_pause`
*   `btn_exit`
*   ...
**请不要修改这些名字！**
Python 代码是靠这些名字来找到按钮并给它们绑定功能的。如果您把 `btn_pause` 改成了 `button1`，点击暂停就没有反应了。

---

## 4. 保存与生效
1.  在 Designer 里修改完后，按 `Ctrl + S` 保存。
2.  **不需要** 重新编译任何东西。
3.  直接运行 `main.py`，您的修改就会立即生效！
